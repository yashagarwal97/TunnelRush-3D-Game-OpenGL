## WebGL Tunnel Rush
A 3D Graphics game developed in WebGL, similar to the game "Tunnel Rush".

# Controls
- a => move left
- d => move right
- w => jump
- g => change display to grayscale
- f => to toggle flashing

# About the game
- Score as much as possible.
- Multiple obstacles
- Texture and lighting on tunnel.
- Flashing.
- Grayscale toggling.
